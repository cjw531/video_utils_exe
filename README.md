# video_utils_exe
GUI Audio & Video extract and merge for Windows 10/11 using `ffmpeg` package.
Download YouTube videos in full.

## Dependencies & Versions
1. ffmpeg: `ffmpeg-7.1.1-essentials_build`
2. yt-dlp: `yt-dlp 2026.03.17`

## Development Instruction
1. Install PyInstaller
```
pip install pyinstaller
```

2. CustomTkinter (UI)
```
pip install customtkinter
```
